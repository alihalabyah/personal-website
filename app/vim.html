My Vim Cheat Sheet

I'm trying to learn vim so every time I learn a new command I'll post it here.

Command to insert mode

a
Append text after cursor position
A
Append text to end of current line
i
Insert text before cursor position
I
Insert text at the start of the current line
j
Move cursor down one line
0
Move cursor to beginning of current line
$
Move cursor to end of current line
w
Move cursor forward one word
b
Move cursor back one word

Command mode

k
Move cursor up one line
j
Move cursor down one line.
h
Move cursor left one character
l
Move cursor right one character
